**Option A: From Vercel**

2. Click "Import Git Repository"
3. Select `uplinkedassitant/simple-token-platform`
4. Configure environment variables
5. Deploy!

**Option B: From GitHub**
1. Go to your repository on GitHub
2. Click "Add file" → "Create new file" (if you want to add README later)
3. Or use Vercel's GitHub integration

### 3. Configure Environment Variables on Vercel

After deploying to Vercel, add these in Settings → Environment Variables:

```
NEXT_PUBLIC_TOKEN_MINT=FY4pnmQ8yQq94jUwEfjnhy93D14BNV6Q5sKSs5Sn8Grw
NEXT_PUBLIC_TOKEN_NAME=Your Token Name
NEXT_PUBLIC_TOKEN_SYMBOL=TOKEN
NEXT_PUBLIC_TOKEN_DECIMALS=9
NEXT_PUBLIC_POOL_ADDRESS=D1ZN9Wj1fRSUQfCjhvnu1hqDMT7hzjzBBpi12nVniYD6
NEXT_PUBLIC_NETWORK=devnet
NEXT_PUBLIC_RPC_URL=https://api.devnet.solana.com
```

### 4. Add SOL Liquidity

Before trading works, add SOL to your pool:
1. Go to https://app.meteora.ag
2. Switch to Devnet
3. Find pool: `D1ZN9Wj1fRSUQfCjhvnu1hqDMT7hzjzBBpi12nVniYD6`
